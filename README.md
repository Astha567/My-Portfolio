# My Portfolio

A simple, clean personal portfolio site built with plain HTML, CSS, and JavaScript — no framework, no build step.

## Files
- `index.html` — page structure and content
- `style.css` — all styling
- `script.js` — mobile menu + the "currently learning" rotator

## Before you deploy — customize these
- [ ] Replace "Your Name" everywhere (title tag, hero, footer)
- [ ] Fill in your field of study, school, and interests in the About section
- [ ] Replace the 3 placeholder project cards with real (or planned) projects
- [ ] Update the skill tags to match what you actually know
- [ ] Update the email, GitHub, and LinkedIn links in the footer
- [ ] Edit the `learningTopics` array in `script.js` to reflect what you're really learning

## Run it locally
Just open `index.html` in your browser — no server needed for a static site like this.

## Deploy it
See the guided steps from Claude, or in short:
1. Push this folder to a GitHub repository
2. Connect the repo to Netlify or Vercel (or enable GitHub Pages)
3. Your site auto-deploys and gets a live URL
